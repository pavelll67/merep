Калькулятор
Простое веб-приложение калькулятора, созданное на Go.

Возможности
Базовые арифметические операции: сложение, вычитание, умножение, деление

Чистый и удобный интерфейс

Поддержка десятичных чисел

Обработка деления на ноль

Технологии
Backend: Go

Frontend: HTML, CSS, JavaScript

База данных: PostgreSQL (опционально, для хранения истории вычислений)
